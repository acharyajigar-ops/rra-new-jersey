# Rajasthan Royals Academy New Jersey — Website

Static marketing site for RRA New Jersey (youth cricket academy, Cranbury, NJ).

## Pages
- index.html — Home
- programs.html — Programs (Foundation / Development / Performance)
- royals-way.html — The Royals Way (philosophy & pathway)
- coaches.html — Coaches
- teams.html — Teams
- register.html — Book a free trial / contact

Each file is fully self-contained (fonts and scripts inlined) and works offline.

## Deploy on GitHub Pages
1. Create a repository and upload all files in this folder to the repo root.
2. Repo Settings → Pages → Source: "Deploy from a branch" → branch `main`, folder `/ (root)`.
3. Your site will be live at https://<username>.github.io/<repo>/

`index.html` is the entry point. Internal links are relative, so it also works by opening index.html directly.
